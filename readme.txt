
# 会話テキスト置換アプリ（replace_app.py）

> テキストファイルの発言者名ハイライトやカスタムリンク変換を、簡単操作で行えるGUIツールです。

---

## 主な機能

- 発言者名（例: シェリー「…）の先頭数文字を指定色でハイライト
- 独自リンク記法（例: `{id,表示名,参照}`）を所定の形式に自動変換
- 変換結果を新しいテキストファイルとして保存

---

## 必要ファイル

- `replace_app.py` : 本体スクリプト
- `speakers.csv` : 発言者名・色・ハイライト文字数の定義ファイル

※ `replace_app.exe` でも実行可能（この場合、speakersは固定です）

---



## 使い方

1. 以下いずれかの方法でアプリをフォルダに用意
	- `replace_app.py` と `speakers.csv` を同じフォルダに置く
	- または `replace_app.exe` をフォルダに置く（この場合 `speakers.csv` は不要）
2. 以下いずれかの方法で起動
	- `replace_app.py` をダブルクリック
	- コマンドプロンプトやPowerShellで `py replace_app.py` を実行
	- `replace_app.exe` を起動
3. 「入力ファイル選択...」ボタンで変換したいテキストファイル（.txt）を選択
4. 「実行」ボタンを押すと、同じ場所に「元ファイル名_output.txt」が生成される
5. ステータス欄やダイアログで処理結果を確認

---

## ファイル仕様

### speakers.csv

発言者名ごとに、色（16進カラーコード）とハイライトする文字数を指定します。


```csv
name,color,length
シェリー,6495ed,3
レイア,ff7f50,3
...（必要に応じて追加）
```


### 入力ファイル例

```txt
エマ「ちょっと待って、それはおかしいよ！」
シェリー「もっと{kawaii,誉めて,賛成 かわいい大好き！えらいねすごいね天才！}ください～！」
```

### 出力ファイル例

```txt
「《color:#ff69b4》ちょっ《/color》と待って、それはおかしいよ！」
「《color:#6495ed》もっと《/color》《link:#kawaii》《red》誉めて《/red》《/link》《ref》賛成 かわいい大好き！えらいねすごいね天才！《/ref》ください～！」
```

---

## 注意事項

- 入力ファイルの文字コードはUTF-8を推奨します。
