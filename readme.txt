# 会話テキスト置換アプリ（replace_app.py）
テキストファイルを自動変換し、発言者名のハイライトやカスタムリンクの置換を行うGUIツールです。

## 主な機能
- 発言者名（例: シェリー「…）の先頭数文字を指定色でハイライト
- リンク（例: {id,表示名,参照}）を所定の形式に変換
- 変換結果を新しいテキストファイルとして保存

## 必要ファイル
- `replace_app.py` : 本体スクリプト
- `speakers.csv` : 発言者名・色・ハイライト文字数の定義ファイル

## 使い方
1. `replace_app.py` と `speakers.csv` を同じフォルダに置きます。
2. `replace_app.py` をダブルクリック、または `py replace_app.py` で実行します。
3. 「入力ファイル選択...」ボタンで変換したいテキストファイル（.txt）を選びます。
4. 「実行」ボタンを押すと、同じ場所に「元ファイル名_output.txt」というファイルが生成されます。
5. ステータス欄やダイアログで処理結果を確認できます。

## ファイル仕様
### speakers.csv
- 発言者名ごとに、色（16進カラーコード）とハイライトする文字数を指定します。
- 例:
  ```csv
  name,color,length
  シェリー,6495ed,3
  レイア,ff7f50,3
  ...
  ```
※ヒロ、ノア、ハンナ、シロのカラーはわからなかったので書いておりません。適宜追加してください。

### 入力ファイル（例）
```
レイア「さあ、劇を始めようじゃないか」
シェリー「もっと{kawaii,誉めて,シェリーちゃんかわいい大好き、えらいねすごいね天才！}ください～！」

```

### 出力ファイル（例）
```
「《color:#ff7f50》さあ、《/color》劇を始めようじゃないか」
「《color:#6495ed》もっと《/color》《link:#kawaii》《red》誉めて《/red》《/link》《ref》シェリーちゃんかわいい大好き、えらいねすごいね天才！《/ref》ください～！」
```

## 注意事項
- `speakers.csv` が見つからない場合、エラーになります。必ずスクリプトと同じ場所に置いてください。
- 入力ファイルの文字コードはUTF-8を推奨します。
